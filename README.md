# CMPA

Desktop application built with Python and Qt (PySide6).

## Setup

1.  Create and activate the conda environment:
    ```bash
    conda create -n CMPA python=3.11
    conda activate CMPA
    ```

2.  Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```

## Running the application

```bash
python src/CMPA/main.py
```
